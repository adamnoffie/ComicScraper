============================================
=  Comic Stripper
============================================

Configuration file is Comics.txt, which needs to be in the same directory as ComicStripper.exe.
It should be pretty self documenting. The project is setup to copy this file to the target directory when building the project.

ComicStripper.sln/csproj -- Visual Studio 2008 solution/project files
ComicStripper_vs2010.sln/csproj -- Visual Studio 2010 solution/project files

Requires:   .NET 3.5 Framework
            Visual Studio 2008 or 2010
